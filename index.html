Through an AJAX request to https://api.github.com/users/Elasticated-Shoe/repos, returns a JSON response. There are several other URLs that you can do a GET request for to get other information. For example, some people using GitHub Pages will store blog posts in the issue tracker (there is no backend in GitHub Pages, it is a workaround to store posts and comments for their sites), they will fetch these issues through the GitHub API. The public API is rate limited by IP adress and too many requests will cause a temporary timeout. To get around this you can use a query string to reduce the amount of date you are returning (like specifying a date range), store the result as a JSON string in localstorage and authenticate with a token to raise the limit.

For more information go to the GitHub API documentation page. class: footspace-2 text-bold ---
